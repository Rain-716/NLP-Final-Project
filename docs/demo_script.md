# 答辩演示脚本

## 1. 架构与状态（30 秒）

说明本地模型只处理代码，ChatECNU 只处理知识增强；展示模型、索引版本和 API 状态。

## 2. 代码解析与问答（2 分钟）

使用流水线 RAW 示例，先生成结构化解析，再追问 `rd == 0` 的意义。强调回答依据具体表达式，且没有调用外部 API。

## 3. 中文注释（45 秒）

使用 RISC-V 译码代码生成注释，展示 opcode、funct3、funct7 的位段含义。

## 4. 知识问答（2 分钟）

询问 Cache 地址拆分。展开证据区，核对 `[S1]` 与来源、章节、行号一致，再说明 ChatECNU 负责重排与组织答案。

## 5. 术语解释与降级（1 分钟）

解释 forwarding、stall、bubble。随后用未配置 API 的测试配置说明：系统只返回本地证据并标注未完成增强。

## 6. 评测（45 秒）

展示 Recall@5、MRR、nDCG@5 和领域外问题误召回；强调 Mock API 测试不等于真实模型质量，人工量表另评估正确性、代码忠实度、引用忠实度、完整性和可读性。
